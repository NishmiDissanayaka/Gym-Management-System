# Gym-Management-System
Gym Management System - IT304082 Database Implementation Final Assignment. Developed using PHP and MySQL following a 3-Tier Architecture.

## GTA bar chart race (1080p)
- Script: `GTA_BarChartRace_1080p.py`
- Install dependencies:
  - `pip install pandas matplotlib bar_chart_race`
- Ensure FFmpeg is installed and available in `PATH` (`ffmpeg -version`)
- Run:
  - `python GTA_BarChartRace_1080p.py`
- Output: `GTA_BarChartRace_1080p.mp4` (axis/ticks are now in **thousands of copies (K)** for smoother detail).
